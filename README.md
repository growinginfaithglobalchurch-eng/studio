# Faith Connect Global

This is a Next.js application built with Firebase Studio, designed to be a dynamic digital platform for a global faith community.

## ‚ú® Features

- **Interactive Dashboard:** A central feed with stories and community updates.
- **Community & Growth:** Connect with members, join groups, and access mentorship.
- **Equipping Centers:** A full Bible school with courses on revelation, healing, prophecy, and more.
- **Warfare & Governance:** Dedicated sections for strategic prayer, including the War Room and Courts of Heaven.
- **Live Content:** Access live ministry, radio, podcasts, and TV studio broadcasts.
- **Admin Dashboard:** Comprehensive tools for managing content, users, and platform settings.

## üöÄ Getting Started

To run the project locally, follow these steps:

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Set up environment variables:**
    Create a `.env` file in the root of the project and add your Firebase and other API keys.

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
