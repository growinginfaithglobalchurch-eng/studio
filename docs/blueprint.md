# **App Name**: Faith Connect Global

## Core Features:

- User Authentication: Secure user authentication via email and social login, with role-based access (Admin, Leader, User).
- Devotionals: Provide daily devotionals and scriptures in audio, video, and text formats.
- Prayer Requests: Allow users to submit prayer requests and join prayer rooms for community support.
- Ministry Content: Enable partner churches/ministries to upload content and schedule events. Implement a tool that uses AI to recommend suitable content to each user, according to their prayer requests and viewing habits.
- Global Networking: Facilitate global networking and collaboration between ministries and users.
- Live Sessions: Enable live streaming of teachings and prayer sessions with on-demand replay.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to evoke trust, faith, and stability.
- Background color: Very Light Blue (#E8EAF6) to create a clean and peaceful environment.
- Accent color: Gold (#FFC107) for highlights, CTAs, and important information.
- Body font: 'PT Sans', a humanist sans-serif for clear and accessible text.
- Headline font: 'Playfair', a modern sans-serif with high contrast, paired with PT Sans.
- Use clean, modern icons to represent various sections and actions within the app.
- Clean, grid-based layout with clear content sections and intuitive navigation.
- Subtle transitions and animations to enhance user experience and provide feedback.
